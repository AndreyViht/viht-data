# Grand Help Viht

Красивое и полезное расширение для админ-панели GRND, которое ускоряет работу модерации и делает ежедневные действия удобнее.

## Авторы

- Проект сделал **Andrey Viht**
- При поддержке **[@vihtvpnbot](https://t.me/vihtvpnbot)**

## Что умеет плагин

- Добавляет удобный виджет статистики прямо на странице.
- Показывает онлайн админов и лидеров в аккуратном уведомлении.
- Копирует список онлайна в буфер обмена.
- Упрощает рутинные действия в админке.
- Работает в связке с вспомогательными данными проекта.
