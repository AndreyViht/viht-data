# Grand Help Viht

Chrome extension for the GRND admin panel that helps moderators work faster, cleaner, and with better visibility.

## Credits

- Created by **Andrey Viht**
- With support from **[@vihtvpnbot](https://t.me/vihtvpnbot)**

## What This Plugin Does

- Adds a floating stats widget on GRND admin pages.
- Shows online admins and leaders in a clean popup.
- Copies online lists to clipboard for quick sharing.
- Improves moderation workflow with extra UI helpers.
- Integrates with external automation/bot flows used by the project.

## Core Idea

This project focuses on practical moderation speed:

- less manual searching,
- faster status checks,
- cleaner operator actions,
- better awareness of who is online and what is happening.

## Notes

- Built for `grnd.gg` and `grnd.game` admin routes.
- Keep extension and helper data updated for best results.
